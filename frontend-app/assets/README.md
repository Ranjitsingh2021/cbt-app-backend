# Assets Directory

## Required Assets

You need to create these placeholder images (or Expo will generate them):

1. **icon.png** - App icon (1024x1024)
2. **splash.png** - Splash screen (1284x2778)
3. **adaptive-icon.png** - Android adaptive icon (1024x1024)
4. **favicon.png** - Web favicon (48x48)

## Quick Solution

Expo can generate these automatically. Just run:

```bash
npm start
```

Expo will create placeholder assets if they don't exist.

Or create simple colored squares:
- icon.png: 1024x1024, solid color #6366f1
- splash.png: 1284x2778, solid color #6366f1
- adaptive-icon.png: 1024x1024, solid color #6366f1
- favicon.png: 48x48, solid color #6366f1

The app will work without these, but you'll see Expo defaults.
